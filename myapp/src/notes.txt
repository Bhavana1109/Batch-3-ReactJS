what is virtual DOM?
-Virtual DOM is a virtual representation of a UI which is kept in a memory later synced(mapping) with 
real DOM with the help of a library called "ReactDOM".

-The updation of real dom with virtual DOM.

-The entire process of updating the realDOM with the help of VDOM is called as"Reconcialities"
